# OpenHands 实战项目指南

## 概述

本指南包含15个实战项目，涵盖OpenHands的所有核心模块。通过完成这些项目，你将全面掌握OpenHands的架构、功能和使用方法。

项目按难度递增排列，建议按顺序完成。每个项目都包含：
- 学习目标
- 前置知识
- 实战步骤
- 验证方法
- 扩展挑战

---

## 项目1：Core模块 - 配置系统实战

### 学习目标
- 理解OpenHands的配置系统
- 掌握日志系统的使用
- 学会自定义配置参数

### 前置知识
- Python基础
- TOML配置文件格式

### 实战步骤

#### 1.1 探索配置系统
```bash
# 查看配置模板
cat config.template.toml

# 探索核心配置代码
find openhands/core/config -name "*.py" | head -5 | xargs cat
```

#### 1.2 创建自定义配置
创建一个自定义配置文件：
```bash
cp config.template.toml my_config.toml
```

修改配置参数：
- 设置不同的LLM模型
- 调整日志级别
- 配置运行时参数

#### 1.3 实现配置验证器
在 `openhands/core/config/` 下创建一个配置验证模块，验证：
- 必需参数是否存在
- 参数值是否在有效范围内
- 依赖关系是否满足

#### 1.4 日志系统实战
创建一个日志测试脚本，演示：
- 不同级别的日志输出
- 结构化日志记录
- 日志过滤和格式化

### 验证方法
- 配置文件能正确加载
- 验证器能捕获无效配置
- 日志输出符合预期格式

### 扩展挑战
- 实现配置热重载功能
- 添加配置加密支持
- 创建配置管理Web界面
